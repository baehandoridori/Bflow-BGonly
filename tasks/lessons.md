## 프로젝트 설정

### 2026-02-19: 초기 세팅

- **데이터 저장 방식**: localStorage 대신 파일 기반 저장 사용해야 함
  - 이유: 앱이 공유 드라이브(G:\)에서 실행되므로, 각 사용자 PC의 localStorage는 공유 불가
  - 해결: `users/{USERNAME}/` 폴더에 JSON 파일로 저장
  - 사용자 식별: `process.env.USERNAME` 또는 `os.userInfo().username`

- **한글 경로 처리**: 배포 경로에 한글이 포함됨
  - `G:\공유 드라이브\JBBJ 자료실\한솔이의 두근두근 실험실\Bflow-BGonly`
  - Node.js `path` 모듈은 한글 경로를 잘 처리하지만, 하드코딩된 문자열 비교 시 인코딩 주의
  - `fs` 모듈의 `{ encoding: 'utf-8' }` 옵션 항상 명시

- **Bflow 원본**: 절대 수정하지 말 것. 참고 전용.
  - 위젯 시스템 참고: `react-grid-layout`, `ResponsiveGridLayout`
  - 상태 관리 참고: Zustand persist 패턴
  - 테마 참고: 다크/라이트 토글 구조
